# MegaActionPlatformer

🎮 基于虚幻引擎的2D动作平台游戏

独立开发 · 持续更新中

📌 项目简介

本项目是一款基于 Unreal Engine 开发的 2D 动作平台游戏，结合 2D 精灵与 3D 场景，实现了 2D/3D 混合渲染的游戏表现形式。

项目基于 PaperZD 动画系统构建完整角色动画状态机，并实现了角色移动、战斗、关卡交互等核心玩法逻辑，具备完整的游戏框架结构。

✨ 项目特色
🎨 2D / 3D 混合渲染
角色使用 2D 精灵（Flipbook / PaperZD）
场景基于 3D 模型构建，增强视觉层次
🧩 模块化游戏框架
角色控制模块
动画状态机模块
碰撞与物理系统
关卡与交互系统
🕹️ 完整动作系统
移动 / 跳跃 / 下落
攻击与受击反馈
动作状态切换（Idle / Run / Jump / Attack）
🎞️ PaperZD 动画驱动
自定义动画蓝图
状态机驱动角色行为
动画与逻辑解耦
⚙️ 可扩展设计
易于增加新技能、新敌人、新关卡
支持后续扩展 AI / UI / 存档系统
